# 게임 컨셉 — 복셀 변경가능 월드 + 인카운터 턴제 RPG (가제 미정)

> 컨셉 후보 중 **B 노선(복셀 + 인카운터 턴제)** 확정.
> A 노선(타일 재건 RPG)은 안전한 선택이었지만, 크래프팅·하우징의 진짜 깊이와 차별성을 위해 B를 선택.
> 이 문서는 "어떻게 만들지"의 의사결정 기준이 되는 살아있는 컨셉 문서다. 결정이 바뀌면 여기서부터 고친다.

---

## 1. 한 줄 컨셉

> **내가 한 블록씩 깎고 쌓는 복셀 세계에서, 적을 만나면 전용 전장으로 전환되는 턴제 RPG.**

코어 가치 세 가지:
- **변형 가능한 세계** — 채굴·건축·하우징이 단순 메뉴가 아니라 월드 그 자체.
- **신중한 한 수의 전투** — 인카운터 시 별도 그리드로 진입하는 클래식 턴제 (GAS 기반 스킬).
- **HD-2D 톤** — 픽셀 캐릭터의 따뜻한 감성을 복셀 위에 얹는다.

---

## 2. 학습 목표 매핑 (6요소)

| # | 요소 | 게임 내 역할 |
|---|---|---|
| 1 | 복셀 월드 | 메인 월드 표현 (탐험·채집·건축의 무대) |
| 2 | HD-2D | 2D 픽셀 캐릭터 + 셰이딩된 복셀 배경의 통합 톤 |
| 3 | GAS | 전투 스킬·버프·디버프·상태이상 전반 |
| 4 | 턴제 전투 | 인카운터 발동 시 그리드 전장에서 진행 |
| 5 | 비헤이비어 트리 | 전투 적 의사결정 + 거점/필드 NPC 일과 |
| 6 | 크래프팅·하우징 | 코어 루프의 절반 (전투가 나머지 절반) |

> 어느 하나도 컨셉에서 제외하지 않는다 — 충돌이 있으면 우회 설계로 해결.

## 3. 비주얼 톤 — "HD-2D 위에 얹은 복셀" 전략

복셀의 각진 큐브와 픽셀 캐릭터, HD-2D의 깊이감은 자칫 충돌한다. 다음 원칙으로 해소.

- **카메라**: 옥토패스 풍 3/4 시점. 큐브가 정면이 아닌 비스듬한 각도로 보이게 → 윤곽이 부드러워짐.
- **셰이딩**: 복셀 표면에 부드러운 그라데이션·앰비언트 오클루전·소프트 섀도우 적용. "각진 마인크래프트"가 아니라 "각진 디오라마"로 보이게.
- **재질/팔레트**: 채도 낮추고 페인팅 톤 팔레트. 픽셀 캐릭터와 동일한 컬러 그루핑 사용.
- **포스트프로세스**: 약한 블룸·뎁스 페이드·필름 그레인. HD-2D 룩의 핵심.
- **픽셀 캐릭터**: 표준 HD-2D 규격 (큰 픽셀, 빛 받음). 환경 라이트가 캐릭터에 반영되도록 정점 라이팅 또는 라이트 함수.
- **표현 라벨**: "복셀 + HD-2D"보다 **"복셀 베이스에 HD-2D 톤 매핑"** 으로 의도를 명확히 한다.

레퍼런스(분위기만 차용): Octopath Traveler, Triangle Strategy, Eastward, Cube World(구조만).

## 4. 코어 게임플레이 루프

| 시간 | 루프 | 활동 |
|---|---|---|
| 30초 | 짧은 | 이동·채집·블록 변경·소규모 인카운터 |
| 5~10분 | 중간 | 새 지역 진입 → 자원 채취 → 인카운터 1~2회 → 발견 보상 |
| 1세션 (30~60분) | 긴 | 거점 복귀 → 크래프팅·하우징·동료 정비 → 다음 목표 결정 |
| 진행축 | 메타 | 지역 해방 → 신규 재료·블록·동료·스킬 해금 |

## 5. 시스템 설계

### 5.1 월드 — 복셀 시스템

- **청크**: 16×16×16 큐브. 동적 생성·언로드. 수정된 청크만 델타 세이브.
- **블록 타입**: 자연 자원(나무·돌·광물) / 건축 자재(가공된 자재) / 기능 블록(작업대·침대·NPC 슬롯) / 장식.
- **가변성 영역 구분 (1인 개발 안전장치)**:
  - **안전 영역(거점)**: 자유롭게 쌓기·깎기 (하우징).
  - **야생 영역**: 채집 중심, 자유 건축은 제한 (불필요한 월드 침식 방지).
- **상호작용**: 좌클릭(채굴), 우클릭(배치), 도구별로 효율·가능 블록 차등.
- **세이브**: 청크 델타 + 거점 NPC 상태 + 인벤토리.

### 5.2 인카운터 → 턴제 전환

- 필드에서 적의 **감지 범위** 안에 들어가거나, 야생 영역 진입 시 확률 발동.
- 전환: **별도 전장 씬 로드** (단순·안정·1인 개발 친화). 페이크 시점 전환은 후순위.
- 전장: N×M 그리드(예: 8×8). 복셀 elevation을 단순 추상화(높이 0~2단계). 지형 효과는 최소.
- 종료 시 필드 위치로 복귀, 결과(피해·보상·아이템) 동기화.

### 5.3 전투 — GAS 턴제 어댑터

GAS는 본래 실시간용이라 턴제에 맞게 컨트롤 레이어를 둔다.

- **TurnManager**: 액션 큐 관리. 현재 액터에게만 Ability Activation 허용.
- **Ability**: 즉시 발동·즉시 종료가 기본. 긴 AbilityTask 지양 (이동·이펙트는 짧은 wait 후 종료).
- **쿨다운**: `Attribute_ChargeRemaining` 같은 게임플레이 어트리뷰트. 매 턴 시작 시 `GE_TickCooldown` 으로 1씩 차감.
- **상태이상**: `GE_Duration`을 "1턴 = 1 duration tick"으로 매핑. 매 턴 종료 시 TurnManager가 tick.
- **GameplayCue**: VFX·SFX는 Cue로 발사 (시각 표현은 그대로 활용 가능).
- **타깃팅**: 그리드 셀 기준. 어빌리티가 타깃 셀과 사거리·시야 검증.
- **AI 호출**: 적 턴이 오면 TurnManager가 BT 실행 → BT가 Ability 결정 후 발동.

장점: GAS의 Effect/Tag/Cue 자산을 그대로 활용 (버프 스택, 면역, 도트, 정화 등 자연스러움).
비용: 어댑터 1개. 학습 가치 큼.

### 5.4 비헤이비어 트리

| 트리 | 위치 | 핵심 노드 흐름 |
|---|---|---|
| `BT_EnemyTurnAI` | 전투 (적 한 턴 결정) | SelectTarget → ChooseAction(Move/Skill) → CommitAbility |
| `BT_VillageNPC` | 거점 | TimeOfDay → GoToWorkplace → DoTask → ReturnHome |
| `BT_WildMob` | 야생 필드 | Patrol → DetectPlayer → (Pursue/Flee) → TriggerEncounter |

블랙보드 키 예: `TargetActor`, `BestSkillIndex`, `MoveCells`, `HealthRatio`, `TimeOfDay`.

### 5.5 크래프팅

- 채집(블록 파괴 드롭) → 인벤토리 → **작업대에서 레시피 조합**.
- 레시피 카테고리: 도구(곡괭이·도끼·검) / 무기·방어구 / 소모품 / 건축 자재 / 가구.
- 도구 티어 = 채집 가능 블록 티어와 게이팅.
- 레시피 발견: 기본 + 책 + NPC 가르침 + 특정 블록 조합 실험.

### 5.6 하우징 + 거점

- 안전 영역에서 복셀 직접 쌓기로 건축.
- **기능 블록**: 작업대·침대·연구대·창고·NPC 슬롯(동료 거주). 기능 블록은 거점 발전도에 기여.
- **동료(NPC) 운영**: 거점에 거주하는 동료는 BT 일과 수행 + 전투 참가.
- **거점 발전도**: 기능 블록 수·동료 수·자원 비축량에 따라 단계 상승 → 신규 레시피·이벤트 해금.

## 6. 진행축 (테마 TBD)

진행 자체는 테마와 독립적으로 다음 골격으로 운영:
1. 초기 거점 발견 → 기본 채집·건축 학습
2. 인접 야생 영역 → 첫 인카운터 → 기본 전투 학습
3. 첫 동료 합류 → 거점 발전 → 신규 지역 잠금 해제 → ...
4. 지역 보스 → 봉인/장벽 해제 → 신규 재료·블록·스킬

테마는 위 골격에 색을 입히는 일 — 다음 섹션에서 결정.

## 7. 마일스톤 (1인 개발, 보수적 추산)

| M | 기간 | 결과물 |
|---|---|---|
| M1 프로토 | 2~4주 | 복셀 청크 1개 + 픽셀 캐릭 이동·채굴·배치 / HD-2D 톤 첫 시도 |
| M2 인카운터 코어 | 4~6주 | 필드 → 전장 전환 / TurnManager / GAS 스킬 2개 (공격·버프) / 승리·패배 |

| M4 크래프팅·동료 | 3~4주 | 작업대·레시피 5개 / 동료 1명 합류·전투참가·BT 일과 |
| M5 하우징·거점 발전 | 3주 | 기능 블록 + 발전도 단계 1 → 2 |
| M6 수직 슬라이스 | 4주 | 지역 1개 + 보스 1 + 톤 정립 + 세이브·로드 |
